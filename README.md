### 🔍 FNDR — Semantic Memory Engine
> *Capstone 2026 | Rust · Tauri · macOS · LanceDB*

**Privacy-preserving desktop memory system** that indexes your entire digital context with zero screenshot persistence.

- 🦀 Rust capture engine via **ScreenCaptureKit** with adaptive sampling
- 👁️ On-device OCR via **Apple Vision** + local transformer embeddings
- ⚡ Sub-100ms query latency over months of activity via **LanceDB**
- 🧹 65% write reduction via perceptual deduplication
- 🔐 Zero cloud inference. Zero raw image storage.

### 🚨 Min0 — Autonomous Incident Response
> *Hackathon 2026 | Python · FastAPI · Kafka · Pinecone*

**Six-agent autonomous pipeline** that classifies, diagnoses, and remediates production incidents before humans wake up.

- 🏗️ SEV-1 to SEV-4 classification with **LLM confidence scoring** in <5s
- 📚 RAG agent over runbooks, Git history & deployment logs via **Pinecone**
- 🛡️ Safety-constrained remediation with per-action rollback plans
- 🔄 Post-mortem agent writes learnings back → 50-80% MTTR reduction target
- 📊 Ingests **Datadog, Prometheus, OpenTelemetry** telemetry

### 🌐 Omni — Unified AI Agent Ecosystem
> *Hackathon 2026 | TypeScript · Claude API · MCP · React*

**Single natural-language command** that autonomously executes across Gmail, Calendar, Drive, Slack, and GitHub.

- 🔗 Unified via **Model Context Protocol (MCP)** across 5+ services
- 🔄 15-iteration sequential tool-call loop for complex multi-step reasoning
- 📡 **NanoClaw WebSocket** gateway with live tool-call streaming
- ⏰ Cron engine for scheduled autonomous agent tasks
- 🧠 Integrates **FNDR** memory + **Min0** incident response as MCP tools
